Użycie identyfikatora ID

Pojęcie Znaczenie pojęcia uzupełnia uczeń
Co znaczy skrót ID? ID oznacza identyfikator (ang. Identifier).
Jaki jest cel stosowania ID? Służy do jednoznacznego oznaczenia konkretnego elementu na stronie i nadania mu stylu.
Jakie jest znaczenie znaku # Znak # w CSS oznacza odwołanie do identyfikatora (ID).
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W CSS używamy znaku # przed nazwą ID, np. #pochyly. W HTML przypisujemy ID przez id="pochyly".
Uwagi na temat stosowania ID (trzy konieczne wyliczenia) 1. ID może wystąpić tylko raz na stronie.
2. Elementy z ID muszą być unikatowe.
3. Nie powinno się używać tego samego ID wielokrotnie (lepiej użyć class).
